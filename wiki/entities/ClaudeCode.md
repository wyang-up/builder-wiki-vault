## 定义
Claude Code 是一种面向软件开发与终端协作的 AI 编码环境，在这篇文章里它被当作 `gstack` 的主要宿主，用来承载技能调用、计划执行、代码实现、审查与发布流程。

## 关键信息
- 在 Garry Tan 的描述里，Claude Code 的价值不只是代码生成，而是作为一个可加载 skills 的执行框架，把产品、设计、工程、QA、安全和发布等角色串成统一工作流。
- `gstack` 通过 slash commands、浏览器工具、多代理配合和 review pipeline，把 Claude Code 从“单次对话式编码工具”推进成“可持续执行 sprint 的操作环境”。
- 文章同时说明 `gstack` 也可以扩展到 OpenClaw、Codex、Cursor、OpenCode 等宿主，但 Claude Code 仍然是它最完整、最典型的原生使用场景。
